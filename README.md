## 📱 产品介绍

**TipCalc Pro** 是一款精美易用的小费计算器 + AA 制分摊工具，帮助用户在餐厅就餐时快速计算小费和分摊金额。

### 核心功能

- 💰 **小费计算** - 输入金额，自动计算小费 + 总额
- 👥 **AA 制分摊** - 输入总金额 + 人数，自动分摊
- 🎨 **精美界面** - 现代设计，支持深色模式
- 📊 **历史记录** - 保存最近计算
- ⚡ **极速启动** - 打开即用，无需等待

## 🚀 快速开始

### 环境要求

- macOS 12.0+
- Xcode 14.0+
- iOS 15.0+

### 构建项目

```bash
# 打开工程
open src/ios/TipCalcPro.xcodeproj

# 或使用命令行编译
xcodebuild -scheme TipCalcPro \
           -destination 'platform=iOS Simulator,name=iPhone 15' \
           clean build
```
