# 产品需求文档：TipCalc Pro

**版本**: v1.0 (MVP)  
**创建日期**: 2026-03-03  
**负责人**: pm

---

## 1. 产品概述

### 1.1 产品愿景
一款精美易用的小费计算器 + AA 制分摊工具，让用户在餐厅就餐时快速计算小费和分摊金额。

### 1.2 目标用户
- 25-45 岁，有外出就餐习惯
- 经常朋友聚会 AA 制
- 愿意为便利付费 ($0.99-$1.99)

### 1.3 核心价值
- 快速计算小费，避免尴尬
- AA 制分摊，公平简单
- 精美界面，使用愉悦

## 4. 验收标准

### 4.1 计算准确性
- [ ] 小费计算准确 (误差 < $0.01)
- [ ] 分摊计算准确 (误差 < $0.01)
- [ ] 实时计算，延迟 < 100ms

### 4.2 界面要求
- [ ] 启动时间 < 1 秒
- [ ] 支持深色模式
- [ ] 适配所有 iPhone 尺寸

### 4.3 性能要求
- [ ] 无崩溃
- [ ] 内存占用 < 50MB
- [ ] 电池消耗可忽略
