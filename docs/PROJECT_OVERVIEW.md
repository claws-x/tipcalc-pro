## 📱 产品简介

**TipCalc Pro** 是一款精美易用的小费计算器 + AA 制分摊工具，帮助用户在餐厅就餐时快速计算小费和分摊金额。

### 核心特色

- 💰 **小费计算** - 快速计算小费和总额
- 👥 **AA 制分摊** - 多人分摊更简单
- 🎨 **精美界面** - 现代设计，支持深色模式
- 📊 **历史记录** - 保存最近计算记录
- ⚡ **极速启动** - 打开即用，无需等待
